== Description ==

Smartly Slick Out your target page into and out of the visible page with this very useful simple side tab. The pages can contain forms, widgets, posts, menu options, comments, logins, sign-ups, annoying popoup ads etc...

There are a few templates included to get you started. The settings page appears in the Dashboard where you can set all your tab slide options such as page source, size, positioning, autohide, opacity. This plugin is ideal for users who would like to add additional content without cluttering up or leaving the page.

Support Blog: http://ataulswebdesigns.com/smarty-slick-tab-plugin/

== Installation ==

1. Upload the `smarty-slick-tab` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You’re done!
4. Go to Dhasboard -> Settings -> Smarty Slick Tab for more options and settings.
